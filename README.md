# FUTURE_ML_01
📊 Sales Forecasting using Machine Learning

✨ Built as part of Machine Learning Internship – Future Interns

---

🚀 Project Overview

This project focuses on predicting future sales based on historical data using a Machine Learning approach.
The goal is to analyze trends and generate accurate forecasts that can help businesses make data-driven decisions.

---

🎯 Objectives

- Analyze historical sales data
- Build a predictive model
- Forecast future sales values
- Visualize trends for better understanding

---

🧠 Machine Learning Approach

- Used Linear Regression for prediction
- Extracted time-based feature (Month)
- Trained model on historical sales data
- Generated predictions for future months

---

📊 Output & Results

- Successfully predicted future sales values:
  
  - Month 6 → 375
  - Month 7 → 408
  - Month 8 → 441
  - Month 9 → 474

- Visualized results using a line graph showing:
  
  - Actual sales (scatter plot)
  - Predicted trend (line plot)

---

📈 Visualization

The graph clearly shows the upward trend in sales and how the model predicts future growth.
